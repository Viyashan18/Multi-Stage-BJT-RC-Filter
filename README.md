# 📻 Multi-Stage Analog Signal Conditioning Circuit

A multi-stage analog electronics project designed to process a weak and noisy audio signal into a clean and amplified output using RC filters and a BJT amplifier.

---

# 📌 Project Overview

This project demonstrates how analog circuits can improve signal quality through filtering and amplification.  

The system simulates the audio processing section of an AM radio receiver by taking a weak noisy signal and converting it into a cleaner and stronger audio output.

---

# 🎯 Objectives

- Remove unwanted low-frequency and high-frequency noise
- Amplify weak audio signals
- Produce a clean output for headphones or small speakers
- Understand practical analog signal conditioning techniques

---

# ⚙️ System Design

## 🔹 Stage 1 — High-Pass RC Filter

A passive RC high-pass filter is used to remove:

- Low-frequency noise
- DC components
- Power line hum

✅ Cutoff Frequency: **2 kHz**

---

## 🔹 Stage 2 — Common Emitter Amplifier (BJT)

A BJT transistor amplifier is used to:

- Amplify weak signals
- Increase voltage gain
- Improve signal strength

✅ Gain achieved: **≥ 60**

The transistor operates in the **active region** using proper biasing.

---

## 🔹 Stage 3 — Low-Pass RC Filter

A passive RC low-pass filter removes:

- High-frequency noise
- RF interference
- Unwanted signal components

✅ Cutoff Frequency: **12 kHz**
